# Webnovel Plan

## Inputs To Inspect

- `设定集/世界观.md`
- `设定集/创作原则.md`
- `大纲/总纲.md`
- Existing `大纲/` files for nearby volumes or chapters.
- `伏笔记录/伏笔总表.md`
- `人物状态/人物总表.md`
- `章节索引/章节索引.md`

Load only files relevant to the requested scope.

## Planning Outputs

For a volume outline, include:

- Volume premise and promise.
- Main external conflict.
- Protagonist internal movement.
- Key antagonistic pressure.
- Major reveals, reversals, and payoff targets.
- Start state and end state.

For chapter outlines, include:

- Chapter number and working title.
- POV and location.
- Opening hook.
- Scene beats.
- Conflict escalation.
- Reveal or decision.
- Ending hook.
- Continuity notes.
- Foreshadowing setup or payoff.
- Character state changes.

For timelines, include:

- Relative order.
- Time gaps.
- Cause-and-effect links.
- Dependencies between reveals, travel, injuries, resources, and relationship changes.

## Update Records

When plans introduce durable facts, update or propose updates to:

- `章节索引/章节索引.md`
- `伏笔记录/伏笔总表.md`
- `人物状态/人物总表.md`
- Relevant files under `设定集/`

Keep speculative options labeled as options, not canon.
